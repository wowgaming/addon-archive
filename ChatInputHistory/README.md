# ChatInputHistory

A lightweight World of Warcraft 3.3.5a (Wrath of the Lich King) client addon that overrides default edit box behavior to allow **Up/Down Arrow** keys to cycle through previously typed chat messages and GM commands without holding the `Alt` key.

## Installation

1. Download or clone this repository into your WoW client directory:
   `World of Warcraft 3.3.5a/Interface/AddOns/ChatInputHistory`
2. Log in and test!
